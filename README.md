# 🛒 Retail Food Store Analysis

## 📌 Overview

Retail Food Store Analysis is a Business Intelligence project developed using Power BI to analyze the distribution of retail food stores across different geographic locations. The dashboard provides meaningful insights into brand presence, store density, and regional performance, helping support data-driven business decisions.


## 🎯 Objectives

- Analyze retail store distribution across different regions.
- Compare brand presence using interactive visualizations.
- Identify high-density store locations.
- Generate insights to support business expansion and strategic planning.


## 🚀 Features

- Interactive Power BI dashboard
- Geographic store distribution analysis
- Brand-wise comparison
- Regional performance analysis
- KPI cards and dynamic charts
- Interactive filters for data exploration


## 🛠️ Technologies Used

- Power BI
- Power Query
- DAX
- Data Modelling


## 📂 Dataset

The project uses a retail food store dataset containing information such as:

- Store Name
- Brand Name
- City
- State
- Region
- Store Location


## 📊 Dashboard Insights

The dashboard provides insights into:

- Store distribution across different regions
- Brand-wise market presence
- Geographic concentration of retail stores
- High-density business locations
- Regional comparison of store availability

## ▶️ How to Run

1. Clone or download this repository.
2. Install **Microsoft Power BI Desktop**.
3. Open the `Final Project.pbix` file.
4. Explore the dashboard using filters and interactive visualizations.
